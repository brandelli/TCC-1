Python é uma linguagem de programação de alto nível, interpretada, interativa e orientada a objetos.
Sua principal caracteristica é o poder de processamento, junto à uma sintaxe muito clara e de fácil compreensão.
Ela é vista como uma linguagem de programação de propósito geral, sendo assim ela pode ser utilizada para diferentes classes de problemas.
Além disso ela vem com uma vasta biblioteca padrão, que pode ser utilizada em inúmeras áreas, tais como processamento de textos, protocolos de internet, interface para sistemas operacionais, entre outras.

A motivação para a escolha desta linguagem é a curva de aprendizado acessivel para o tempo necessário de desenvolvimento da tarefa, e também conta com uma comunidade muita ativa, e atenciosa quanto à auxiliar com questões quanto à linguagem em si, ou as diversas bibliotecas que existe.
